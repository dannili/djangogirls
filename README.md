# A Simple Django Blog Post Project

## Features

1. User Log in
2. Add/Change blog posts
3. Display the post list


## Get It Running

1. Create a superuser
    ```sh
    python manage.py createsuperuser
    ```
2. Running on http://127.0.0.1:8000/
    ```sh
    python manage.py runserver
    ```
